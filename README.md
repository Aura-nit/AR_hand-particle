# Hand Particle Web App

ARハンドパーティクルWebアプリケーションである．
MediaPipeを使用して手の動きを認識し，Three.jsを用いてパーティクルエフェクトを生成する．

## 機能

手のジェスチャーに応じて異なるパーティクルエフェクトが発生する．

- **基本操作**:
    - 手を開く: パーティクルが拡散する．
    - 手を閉じる: ブラックホールのようにパーティクルが収束する．

- **ジェスチャー効果**:
    - **指差し (Pointing)**: 赤色のエフェクト (赫/Reversal Red)．収束と放出が混在する．
    - **OKサイン**: 紫色のエフェクト (虚式・紫/Hollow Purple)．強力な引力で高密度の球体を形成する．
    - **両手検出**: 青色のエフェクト (蒼/Azure)．両手の中間地点でパーティクルが生成され，収束する．
    - **領域展開 (人差し指と中指を交差)**: 背景が黒くなり，白いパーティクルが舞う (無量空処/Domain Expansion)．

## 技術スタック

- **フレームワーク**: Vite
- **言語**: TypeScript
- **グラフィックス**: Three.js
- **ビジョン**: MediaPipe Tasks Vision

## セットアップと実行

プロジェクトの依存関係をインストールする．

```bash
npm install
```

開発サーバーを起動するには以下のコマンドを実行する．

```bash
npm run dev
```

ビルドを行うには以下のコマンドを実行する．

```bash
npm run build
```

プレビュー（ビルド後の確認）:

```bash
npm run preview
```
